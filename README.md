# OpenPecha Parallel Corpus

This corpus contains all the multilingual texts aligned to Tibetan texts (bo) at the sentence-level.  

## Overview
- **2,08,736** Tibetan segments 
- **4,611** files 
- Files from [Lotsawa House](https://www.lotsawahouse.org/), ...
